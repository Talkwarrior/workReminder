과제 독촉기
2305 백종헌
1. 필요성
대구과학고의 특징 중 하나는, 과제가 단기간에 몰려 있다는 것이다. 주제, 제출 기한, 각종 중간 보고서와 모둠활동 여부까지 다양한 변수들이 집합된 과제들을 모두 기억하기란 쉽지 않다. 실제로 작년에는 과제의 존재를 잊고 있다가 소위 ‘벼락치기’로 처리하는 학생들을 많이 보았다.
그래서 자신의 과제를 모두 파악하고, 제출 기한이 임박했을 때 윈도우 알림을 보여주는 프로그램을 구상하였다.

2. 기능
기본적인 기능은, 과제를 저장해 놓았다가 사용자가 원할 때 보여주는 것이다. 구글 클래스룸 API를 사용하여, 과제 정보를 매시간 업데이트하여 저장한다. 그리고 과제 정보를 정리하고, 스케줄러에 실행을 예약한 후 종료한다.
스케줄러에 저장된 일정에 따라 팝업창이 뜬다. 팝업창에는 과제에 대한 정보가 담겨 있으며, 이때 과제 해결 여부를 입력 받을 수 있다.

3. 필요 모듈
google-api-python-client, google-auth-httplib2, google-auth-oauthlib
: 구글 클래스룸 API
schedule
: 스케줄 작성
pyqt5
: GUI 팝업 구성, 메인 윈도우 구성
JSON
: 과제 저장 시 사용

4. 알고리즘
프로그램은 두 부분으로 나뉜다. 메인 윈도우와 백그라운드 프로세스이다.
메인 윈도우는 사용자가 직접 사용하는 창이다. 과제 목록 보기, 과제 삽입/수정/삭제가 가능한 프로그램이다. 프로그램이 종료될 때 과제 목록을 덮어씌운다.
백그라운드 프로세스는 매시간 과제 목록을 업데이트한다. 그리고 과제의 제출기간을 보고 제출 일주일 전부터 사용자에게 안내 팝업을 보여줄 것이다.
구글 클래스룸 API를 통해 얻어 온 과제 정보를 기존의 데이터와 비교하는 알고리즘이 필요하다. 이는 과제와 입력 정보를 정렬하여 저장해 놓고, 첫 부분부터 비교하여 부족한 부분을 채우는 형식으로 해결할 것이다.
