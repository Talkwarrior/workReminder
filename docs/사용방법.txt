pip를 이용해 필요한 패키지(requirements.txt)를 설치합니다.python -m install -r requirements.txt최상위에 있는 두 개의 파일(autoReminder, main_exec)을 모두 실행해야 하는데,실제 편의를 위해서는 윈도우의 작업 스케줄러를 열어 컴퓨터 시작 시 pythonw로 autoReminder.py를 실행하는 스케줄을 생성하는 것이 좋습니다.main_exec.py를 실행하면 과제들을 수정할 수 있으며, autoReminder는 자동으로 윈도우 알림을 보낼 것입니다.https://github.com/Talkwarrior/workReminder에는 최신 소스를 올려놓았으며, 인스톨러를 release탭에 생성하였으니실용적인 버전은 거기에서 다운받을 수 있습니다.현재 버전은 1.4.0입니다.
